## Setup

- install dependencies

```bash
yay -S astal-io aylurs-gtk-shell-git wireplumber libgtop bluez bluez-utils networkmanager dart-sass wl-clipboard upower gvfs gtksourceview3 libsoup3zoom python python-gpustat brightnessctl pywal pacman-contrib power-profiles-daemon grimblast wf-recorder hyprpicker hyprsunset swww libastal-meta
```

- generate types
```
cd shoji-bar
ags types -u -d ./
```

## Usage
`ags run ./app.ts`
